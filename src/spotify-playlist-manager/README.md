# Spotify Playlist Manager

A C# tool to transfer playlists between Spotify accounts using the Spotify Web API.

## Features
- Authenticate with OAuth 2.0
- List user playlists
- Transfer playlists from one account to another
- Merge playlists

## Prerequisites
- .NET 8.0 SDK
- Spotify Developer account (for API credentials)

## Setup
1. Clone the repo
2. Set environment variables: `SPOTIFY_CLIENT_ID`, `SPOTIFY_CLIENT_SECRET`
3. Run `dotnet restore`
4. Run `dotnet run`

## Usage
```
dotnet run --help
```
