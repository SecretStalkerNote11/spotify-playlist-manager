## ❓ FAQ

**Q: Is this safe to use with my Spotify account in 2026?**  
A: Yes. The tool uses Spotify's official OAuth API and never stores your password. Spotify allows playlist management via API by design. With reasonable use (under 100 playlist transfers per day), the risk of any action against your account is extremely low.

**Q: How often is this tool updated?**  
A: Typically within 48 hours of any Spotify API changes. Check the Releases page for the latest version. The 2026 release is fully compatible with current Spotify API v1 endpoints.

**Q: A transfer failed halfway through—what do I do?**  
A: The tool automatically resumes from the last successful track. If you encounter persistent failures, check the log file (`logs/transfer_YYYY-MM-DD.log`) for specific error codes. Common fixes include re-authenticating or waiting 60 seconds between retries.

**Q: Can I transfer playlists to a brand new Spotify account?**  
A: Yes. The destination account only needs to exist and be authenticated. New accounts work fine, though Spotify may impose temporary rate limits for the first 24 hours.

## 🛡️ Safety & Best Practices

- Use a dedicated Spotify account for testing if you're transferring large libraries
- Avoid transferring more than 200 playlists in a single session
- The tool includes built-in delays to respect Spotify's rate limits
- No data is transmitted to any server except Spotify's official API endpoints
